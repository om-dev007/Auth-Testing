# 🔐 Authentication Learning Project

A simple full-stack authentication project built for learning purposes.
It demonstrates user signup and login using a React frontend and Express backend connected to MongoDB Atlas.

---

## 🌐 Live

Frontend: https://backend-auth-testing.vercel.app                                                                                                                                        
Backend: https://auth-backend-yl53.onrender.com

---

## 🧱 Tech Stack

**Frontend**

* React
* React Router DOM
* Fetch API
* Vercel

**Backend**

* Node.js
* Express.js
* MongoDB Atlas
* Render

---

```
frontend/
```

## 🔄 Flow

User → React UI → Backend API → MongoDB → Response → UI
