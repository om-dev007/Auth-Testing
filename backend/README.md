# 🔐 Auth Backend

Express backend providing signup and login APIs connected to MongoDB Atlas.

---

## 🌐 Base URL

https://auth-backend-yl53.onrender.com

---

## 🧱 Stack

* Node.js
* Express
* MongoDB Atlas
* CORS
* cookie-parser (installed, not used yet)

---

## 📂 Structure

```
src/
  controllers/
  models/
  routes/
app.js
server.js
```

---

## 🔌 Endpoints

**Register**

```
POST /api/auth/register
```

**Login**

```
POST /api/auth/login
```
